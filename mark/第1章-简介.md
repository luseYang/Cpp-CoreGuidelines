# 简介

- _C++ Core Guidelines 的目标读者群是 **所有 C++ 程序员**，包括可能考虑使用 C 语言的程序员。_

- C++ Core Guidelines 的规则倡导现代 C++，旨在实现更统一的风格。

- Guidelines 并不要求连续阅读，也不会替代教材。
